Это мини-магазин, где можно:

создать товар (Item) через админку
открыть страницу товара
нажать кнопку Buy
перейти на Stripe Checkout и оплатить (в тестовом режиме)
    Как он работает
Пользователь заходит на /item/1
Видит описание товара и цену
Нажимает Buy
Backend создаёт Stripe Session
Пользователь попадает на страницу оплаты Stripe
После оплаты:
/success/ - успешная оплата
/cancel/ - отмена
    
Что внутри проекта
Django модель Item (товары)
API /buy/<id>/ для создания платежа
HTML страницы через Django templates
Stripe Checkout для оплаты
базовые страницы success/cancel
